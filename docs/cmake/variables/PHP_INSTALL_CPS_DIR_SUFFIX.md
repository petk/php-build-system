# `PHP_INSTALL_CPS_DIR_SUFFIX`

* Default: `PHP`

The name of the directory inside the `lib/cps/` where to install PHP CPS files
(`PHP.cps`). For example, `PHP/8.6` to specify version or other build-related
characteristics and have multiple PHP versions installed. If absolute path needs
to be set, configure `CMAKE_INSTALL_LIBDIR` along with this path.

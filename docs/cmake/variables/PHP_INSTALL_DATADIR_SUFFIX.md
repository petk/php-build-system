# `PHP_INSTALL_DATADIR_SUFFIX`

* Default: `php`

The name of the directory inside the `CMAKE_INSTALL_DATADIR` where to install
PHP read-only architecture-independent data. For example, `php/8.6` to specify
version or other build-related characteristics and have multiple PHP versions
installed. If absolute path needs to be set, configure `CMAKE_INSTALL_DATADIR`
instead.

# `PHP_INSTALL_DOCDIR_SUFFIX`

* Default: `PHP`

The last part of the `CMAKE_INSTALL_DATADIR` directory name (the name of the
directory inside the `share/doc/`) where to install
PHP documentation-related files. For example, `PHP/8.6` to specify version or
other build-related characteristics and have multiple PHP versions installed. If
absolute path needs to be set, configure `CMAKE_INSTALL_DOCDIR` instead.
